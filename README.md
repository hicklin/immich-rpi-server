This project aims to provide a low-cost and simple way to set up and run an [Immich](https://immich.app/) instance, empowering you to move away from big tech without requiring a computer degree. Having said that, you will be running your own server which will require understanding of some basic concepts. This project attempts to provide sufficient documentation and explanations in the from on a [short book](https://hicklin.github.io/immich-rpi-nix/) to get you on your way towards regaining sovereignty over your personal data.
