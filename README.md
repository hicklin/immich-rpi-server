# Immich on NixOS for Raspberry Pi

Badges...

This project aims to provide a cheap and simple way to set up and run an [Immich](https://immich.app/) instance, empowering you to move away from BigTech without needing a computer degree. This is achieved through the use of NixOs, the declarative Linux Operating System (OS). Having said that, you will be running your own server which will require understanding of some basic concepts. This project attempts to provide sufficient documentation and explanations in the from on a [short book]() to get you on your way on regaining sovereignty over your personal data.
